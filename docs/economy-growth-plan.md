# 金币经济与成长配表 1.0

## 1. 核心规则

| 项目 | 规则 |
| --- | --- |
| 金币来源 | 仅来自局内拾取金币 |
| 额外获取 | 仅允许结算后观看 1 次激励广告，将本局金币翻倍 |
| 非来源项 | 分数不转金币；签到、任务、邮件、离线收益均不产金币 |
| 体力门槛 | 每局消耗 5 体力，当前满体力可玩 6 局 |
| 经济目标 | 不看广告也能稳定升级；看广告能明显提速，但不应成为唯一可行路径 |

## 2. 货币结构建议

建议将当前局内积分 Coin 与成长金币拆开。

| 资源 | 用途 | 获取方式 |
| --- | --- | --- |
| 分数 | 排行、难度、局内反馈 | 现有积分 Coin、穿越障碍、奖励房表现 |
| 金币 | 局外成长升级 | 局内专门的金币拾取物 + 结算广告双倍 |

这样可以避免奖励房和高连锁操作直接冲垮升级经济。

## 3. 单局金币产出目标

### 3.1 玩家体感目标

| 表现层级 | 单局时长 | 不看广告金币 | 广告双倍后 | 目标体感 |
| --- | --- | --- | --- | --- |
| 新手失误局 | 20-35 秒 | 12-18 | 24-36 | 也能攒，失败不至于无收获 |
| 稳定通关局 | 35-60 秒 | 20-28 | 40-56 | 2-4 局能完成一次前期升级 |
| 熟练高分局 | 60-90 秒 | 30-42 | 60-84 | 会产生“这局值得翻倍”的感觉 |
| 高光奖励局 | 90 秒以上/进奖励房 | 45-70 | 90-140 | 成为广告转化的主要触发点 |

### 3.2 单管体力总收益目标

按 30 体力、每局 5 体力计算，一管体力约 6 局：

| 玩家行为 | 单局均值 | 6 局总收益 |
| --- | --- | --- |
| 从不看广告 | 24 金币 | 144 金币 |
| 一半局数看广告 | 36 金币 | 216 金币 |
| 每局都看广告 | 48 金币 | 288 金币 |

设计目标：

- 不看广告：一管体力可完成 2-3 次低阶升级，或 1 次中阶升级。
- 经常看广告：一管体力可完成 3-5 次低阶升级，或 1-2 次中阶升级。

## 4. 局内金币掉落建议

### 4.1 金币拾取物类型

| 类型 | 单枚金币值 | 典型投放 | 设计目的 |
| --- | --- | --- | --- |
| 小金币 | 1 | 普通安全路径、基础奖励 | 保证每局稳定产出 |
| 大金币 | 2 | 较危险路径、障碍边缘、切线位置 | 用风险换更多收益 |
| 金币包 | 3 | 奖励房、极少数高风险奖励点 | 制造高光局与广告动机 |

### 4.2 产出占比建议

| 类型 | 产出占比 | 说明 |
| --- | --- | --- |
| 小金币 | 60% | 保证稳定、保底 |
| 大金币 | 25% | 拉开熟练玩家收益 |
| 金币包 | 15% | 主要集中在奖励房或高风险路线 |

### 4.3 奖励房金币控制

奖励房建议只提供高价值金币包，不再铺满普通成长金币。

| 奖励房表现 | 建议金币产出 |
| --- | --- |
| 普通进入一次 | 15-20 |
| 表现良好 | 20-28 |
| 高光房 | 30-40 |

目标是让奖励房很爽，但不能单次等于 3-5 局常规局收益。

## 5. 广告双倍触发策略

### 5.1 结算页展示信息

结算页双倍按钮必须同时展示以下信息：

| 展示项 | 示例 |
| --- | --- |
| 本局金币 | 本局获得 26 金币 |
| 广告后金币 | 观看广告后可获得 52 金币 |
| 升级关联 | 再获得 14 金币即可升级“冲刺时长 Lv3” |

### 5.2 推荐触发阈值

| 条件 | 是否推荐强提示双倍 |
| --- | --- |
| 本局金币 < 12 | 否，弱提示即可 |
| 本局金币 12-24 | 是，常规提示 |
| 本局金币 > 24 | 是，强提示 |
| 双倍后可立即升级任意内容 | 必须强提示 |
| 进入过奖励房/高光局 | 必须强提示 |

目标不是让玩家局局被迫看广告，而是让玩家在“这局很赚”时主动点双倍。

## 6. 成长线规划

首版建议只开放 6 条成长线，每条 8 级。

| 成长线 | 类型 | 基准值 | 8 级后目标值 | 成本基线 | 设计目的 |
| --- | --- | --- | --- | --- | --- |
| 冲刺时长 | 爽感核心 | 5.0 秒 | 7.4 秒 | 50 | 直接提升高光局频率 |
| 无敌时长 | 容错核心 | 10.0 秒 | 14.8 秒 | 60 | 提升中低水平玩家留存 |
| 奖励房时长 | 高价值成长 | 10.0 秒 | 16.0 秒 | 90 | 强化高光与广告动机 |
| 拾取范围 | 手感成长 | 80 | 128 | 75 | 让玩家明显感到更顺手 |
| 金币倍率 | 经济核心 | 100% | 145% | 110 | 提升长期刷金效率 |
| 急速冲刺时长 | 局外道具强化 | 8.0 秒 | 12.0 秒 | 130 | 强化开局爽点，适合中后期 |

## 8. 玩家升级节奏预期

按单局均值 24 金币、常看广告后单局均值 36 金币估算：

| 成长线 | 全 8 级累计金币 | 不看广告所需局数 | 半数局双倍所需局数 | 每局双倍所需局数 |
| --- | --- | --- | --- | --- |
| 冲刺时长 | 1287 | 54 局 | 36 局 | 27 局 |
| 无敌时长 | 1544 | 65 局 | 43 局 | 33 局 |
| 拾取范围 | 1928 | 81 局 | 54 局 | 41 局 |
| 奖励房时长 | 2314 | 97 局 | 65 局 | 49 局 |
| 金币倍率 | 2827 | 118 局 | 79 局 | 60 局 |
| 急速冲刺时长 | 3340 | 140 局 | 93 局 | 70 局 |

这个节奏意味着：

- 玩家前期能快速点出 2-3 条低阶成长。
- 中期开始自然产生双倍广告动力。
- 高价值成长不会在前几天被迅速毕业。

## 9. 首版推荐解锁顺序

| 开放阶段 | 成长线 |
| --- | --- |
| 开局默认开放 | 冲刺时长、无敌时长、拾取范围 |
| 任意两条成长达到 Lv3 | 奖励房时长、金币倍率 |
| 玩家拥有急速冲刺道具后 | 急速冲刺时长 |

这样可以保证：

- 前期先强化手感和容错。
- 中期再开放经济和高价值房间强化。
- 后期才进入局外道具深挖。

## 10. 实现建议

### 10.1 数据结构

建议至少拆成两张表：

#### GrowthUpgradeDef

| 字段 | 类型 | 说明 |
| --- | --- | --- |
| Id | string | 成长线 ID，如 `dash_duration` |
| Name | string | 显示名称 |
| MaxLevel | int | 最大等级 |
| UnlockCond | string | 解锁条件 |
| ValueType | string | `duration` / `radius` / `percent` |

#### GrowthUpgradeLevel

| 字段 | 类型 | 说明 |
| --- | --- | --- |
| UpgradeId | string | 对应成长线 ID |
| Level | int | 等级 |
| Value | float | 当前级效果值 |
| CostGold | int | 升级消耗 |

### 10.2 结算逻辑

| 步骤 | 规则 |
| --- | --- |
| 1 | 记录本局拾取金币 `runGold` |
| 2 | 结算时先发放 `runGold` |
| 3 | 若玩家观看双倍广告，再额外发放 `runGold` |
| 4 | 同一局只允许领取 1 次双倍 |

### 10.3 结算页文案

| 场景 | 推荐文案 |
| --- | --- |
| 普通局 | 本局获得 `26` 金币，观看广告可追加 `26` |
| 高光局 | 本局高收益，观看广告可额外获得 `42` 金币 |
| 可立即升级 | 再获得 `18` 金币即可升级“冲刺时长”，观看广告可立即升级 |

## 11. 后续版本扩展点

1. 金币拾取路径分层：安全路少、危险路多。
2. 奖励房金币包视觉强化，让玩家更容易把它识别为“必须双倍”的高价值局。
3. 中后期可引入第二货币，但首版不建议同时做。
4. 如果后续要加商城或活动，仍然不要直接送大量金币，否则会冲击广告翻倍价值。
